# Overview

This dataset package has been generated using the Enterprise Synthetic Data Generation Engine (ESDGE), a dedicated subsystem of the AI Product Manager Copilot project.

The datasets are designed to simulate realistic enterprise product management environments and support:

- PostgreSQL relational database
- REST API backend
- Analytics dashboards
- AI Recommendation Engine
- Retrieval-Augmented Generation (RAG)
- Qdrant Vector Database
- Gemini AI Integration

The generated data follows consistent primary keys, foreign key relationships, and realistic business distributions.

# Purpose of Each Dataset

Organizations
Represents enterprise customers using the AI Product Manager Copilot platform.

Workspaces
Logical workspaces within each organization.

Users
Employees, product managers, developers, QA engineers, and administrators.

Projects
Software products managed inside each workspace.

Feedback
Customer feedback collected for AI analysis.

Features
Feature requests extracted from product planning.

Tickets
Engineering implementation tasks.

Sprints
Sprint scheduling and Agile planning.

Analytics Metrics
KPIs used by dashboards and reporting modules.

RAG Chunks
Knowledge chunks prepared for semantic search and AI retrieval.
